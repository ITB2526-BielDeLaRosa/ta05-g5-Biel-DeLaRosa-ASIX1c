PROJECTE 2: SCRIPT D'AUTOMATITZACIÓ (PYTHON)
Tecnologies: Python 3.9, Paramiko (SSH), JSON, Linux Server.

Descripció tècnica


He creat un script en Python dissenyat per automatitzar el manteniment de servidors Linux remots. Aquesta eina estalvia hores de feina manual.


Funcionalitats Clau:

  • Connexió massiva via SSH.
  • Actualització automàtica de paquets (apt-get update).
  • Revisió de logs de seguretat.
  • Sistema d'alertes si un servidor cau.

El codi és modular i permet afegir noves funcions fàcilment mitjançant un fitxer de configuració JSON.
