PROJECTE 2: APP DE PREVISIÓ METEOROLÒGICA
Documentació tècnica: HTML5, SASS, JavaScript (Fetch API), OpenWeatherMap.

Descripció del projecte


Aquesta aplicació web permet consultar l'estat del temps en temps real de qualsevol ciutat del món. Es connecta a una API externa per obtenir dades actualitzades.


Funcionalitats destacades:

  • Cercador de ciutats amb autocompletat.
  • Visualització de temperatura, humitat i velocitat del vent.
  • Icones dinàmiques segons si fa sol, plou o està ennuvolat.
  • Mode fosc automàtic segons l'hora del dia.

El repte principal d'aquest projecte va ser gestionar les peticions asíncrones (AJAX) i processar la resposta en format JSON per pintar-la a la pantalla.
