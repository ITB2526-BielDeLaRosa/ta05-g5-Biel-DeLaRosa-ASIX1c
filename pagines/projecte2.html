PROJECT 2: AUTOMATION SCRIPT (PYTHON)
Technologies: Python 3.9, Paramiko (SSH), JSON, Linux Server.

Technical description


I have created a script in Python designed to automate the maintenance of remote Linux servers. This tool saves hours of manual work.


Key Features:

  • Bulk connection via SSH.
  • Automatic package update (apt-get update).
  • Security log review.
  • Alert system if a server goes down.

The code is modular and allows new features to be added easily via a JSON configuration file.
